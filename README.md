# Project

## Project info

This repository contains the Joben.eu web application.

## How can I edit this code?

Edit locally with your preferred IDE or use GitHub to modify files.

Local development (example):

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm i
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

Build with `npm run build` and deploy the generated `dist/` according to your hosting provider.

## Custom domain

Connect a custom domain via your hosting provider; configuration depends on the chosen host.
